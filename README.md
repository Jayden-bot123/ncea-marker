# NCEA Marking AI

An AI-assisted tool designed to help teachers assess NCEA Level 1, 2, and 3 internal assessments more quickly and efficiently.

The project aims to reduce marking workload by helping teachers review student submissions faster, organise grade decisions, and return feedback to students sooner. By saving time on administrative marking tasks, teachers can spend more time teaching and supporting students in class.

## Goals

* Speed up the marking process for internal assessments
* Support consistent grading against NCEA criteria
* Help teachers return results and feedback faster
* Reduce workload so more time can be spent on learning

## Current Status

In active development.

## Note

This tool is intended to assist teachers, not replace professional judgement. Final grading decisions should always remain with the teacher.
